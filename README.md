# Learning Studio — port

A React/TypeScript port of the legacy `learning-studio-app` (the deployed
single-file bundle lives at `reference/deployed-index.html`, 734 lines).

This project is the source of truth for the deployed app on Lemma at
<https://learning-studio-port.apps.lemma.work>. CI pushes from `main`
redeploy automatically — see `.github/workflows/deploy.yml`.

## Layout

| Path | Purpose |
|---|---|
| `src/` | The actual app (React 19, Vite, TanStack Query, `@lemma/sdk/react`) |
| `reference/deployed-index.html` | The legacy single-file bundle, kept as the porting contract |
| `PORTING.md` | Section-by-section map from bundle → React component |
| `.github/workflows/deploy.yml` | GitHub Actions → `lemma apps deploy` on every push to `main` |

## Develop

```bash
npm ci
npm run dev      # http://localhost:5173
```

`npm run dev` reads `.env.local` (auto-generated by `lemma apps init`)
for `VITE_LEMMA_API_URL`, `VITE_LEMMA_AUTH_URL`, `VITE_LEMMA_POD_ID`.

## Deploy

| Path | Tool |
|---|---|
| Push to `main` on GitHub | auto-deploys via CI |
| Manual | `lemma apps deploy learning-studio-port --pod <pod-id> -y` |

Both rewrite the existing app record. There is no migration step;

## Required GitHub secrets

| Secret | What |
|---|---|
| `LEMMA_TOKEN` | A Lemma personal API token with `app.deploy` grant |
| `POD_ID` | `019efe5e-b261-76be-8124-a686c1d07311` (Lemma pod the app lives in) |

Get `LEMMA_TOKEN` from <https://lemma.work/account/api-tokens>.

## Source archive on Lemma

The pod stores the latest deployed bundle as `source/archive.zip`. See the
live app record with `lemma apps get learning-studio-port --full`.
